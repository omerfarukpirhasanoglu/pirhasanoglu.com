# My Website

Kişisel portfolyo web sitem.  
Next.js, React, TypeScript ve TailwindCSS kullanılarak geliştirilmiştir.

# Live Demo
https://pirhasanoglu.com

# Tech Stack

- Next.js
- React
- TypeScript
- TailwindCSS
- Vercel (Deployment)

# Installation

Projeyi localde çalıştırmak için:

```bash
npm install
npm run dev
```

# Build

```bash
npm run build
```
